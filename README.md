# Weather
Aplicativo de Clima
Este é um aplicativo de clima que busca informações diretamente na API Openweathermap e, de acordo com a descrição do clima da cidade pesquisada, busca uma imagem de fundo em outra API, Unsplashapi.

O aplicativo foi desenvolvido utilizando as dependências jQuery e Axios.

Como utilizar
Para utilizar o aplicativo, basta abrir o arquivo index.html em um navegador web e inserir o nome da cidade que deseja consultar na barra de pesquisa. O aplicativo irá exibir a previsão do tempo atual, além da previsão para os próximos três dias.
